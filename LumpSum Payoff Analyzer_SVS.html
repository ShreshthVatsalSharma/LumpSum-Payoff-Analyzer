How the Quarterly EMI is Computed
Standard amortizing loan formula applied to the financed portion of the plot price.
P = Plot Price × Financed %
r = Annual Rate / 4 / 100 (quarterly)
n = Number of installments
EMI = P × r × (1+r)^n / [(1+r)^n - 1]
Each EMI contains two parts: an interest component (principal × quarterly rate) and a principal component (EMI minus interest). As installments progress, the interest share falls and the principal share rises.
The Lump Sum Decision Problem
Why we are doing this ?

If ADA allows lump sum too early, it forfeits interest income that would have accrued on the unpaid balance. The full interest pool is earned only if all installments run to completion.

If ADA never allows lump sum, it maximises interest income but misses the liquidity benefit of receiving the full outstanding principal at once.

The optimal quarter is where the marginal interest in remaining installments becomes negligible relative to the cash advantage of an immediate lump sum. This platform intends to identify that point precisely.
